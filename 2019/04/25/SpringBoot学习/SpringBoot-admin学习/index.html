1. 什么是Spring Boot Admin

Spring Boot Admin 是一个社区的项目用来监控和管理我们的Spring Boot 应用。应用注册可以通过 Spring Boot Admin Client (Http) 或者使用 Spring Cloud 注册中心去发现（例如Eureka, Consul)。
